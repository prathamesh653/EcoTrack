 🌱 EcoTrack – Smart Environmental Monitoring System

📌 Overview

EcoTrack is a full-stack application designed to monitor, analyze, and track environmental data such as air quality, pollution levels, and sustainability metrics. The system helps users make data-driven decisions to promote a greener and healthier environment.

🚀 Features

 📊 Real-time environmental data tracking
 🌍 Pollution level monitoring (Air, Water, etc.)
 📈 Data visualization with charts and graphs
 🔔 Alerts for critical environmental conditions
 👤 User authentication and role-based access
 📦 Scalable backend with REST APIs
 🐳 Dockerized application for easy deployment

 🛠️ Tech Stacks

#💻 Backend

* Java
* Spring Boot
* REST APIs

# 🌐 Frontend

#⚙️ Installation & Setup

#🔹 Clone the Repository

git clone https://github.com/prathamesh653/EcoTrack.git
cd EcoTrack


# 🔹 Run using Docker (Recommended)

docker-compose up --build

# 🔹 Run Backend Manually

cd backend
mvn spring-boot:run

# 🔹 Run Frontend

cd frontend
npm install
npm start
